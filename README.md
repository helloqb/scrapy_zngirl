# scrapy_zngirl
个人 scrapy 框架学习，以 zngirl 网站练手（毕竟妹子是第一生产力）。

## 运行
1. 爬虫入口 zngirl.py，替换变量 start_urls 为个人主页
2. 运行 main.py 文件

## 结果
当前项目 `images` 目录下，分专辑保存。
